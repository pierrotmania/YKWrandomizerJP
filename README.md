**これは何？**

オリジナルの"Yo-kai Watch Randomizer"を日本語版の妖怪ウォッチで動作できるようにしました。
また、メニュー(UI)も日本語で利用できます。

以下からはオリジナルから翻訳した文章をベースとしています。

**免除責任**

私はレベルファイブとは一切関係ありません。
妖怪ウォッチシリーズを楽しむファンとして、新しい遊び方を共有したいだけです！
ランダマイザーは一人で遊ぶときに楽しむ目的、あるいは新しい遊び方を発見する目的でのみ使用してください。

**ランダマイザーとは？**

「ランダマイザー」は、ゲームの要素をランダム化する仕組みで、ポケモンが広めました。
YKWrandomizerはYo-Kai Watchシリーズ向けのランダマイザーで、このツールを使うと以下のようなゲーム内要素をランダム化できます。
妖怪の種族、ステータス、技、進化など。

**プロジェクトについて**

YKWrandomizerは、いくつかのYo-Kai Watchゲームをランダム化するアプリケーションを作るプロジェクトです。
ツールはC#で書かれており、オープンソースなので誰でも貢献できます。

**対応ゲーム**
- 妖怪ウォッチ(初代) ✅
- 妖怪ウォッチ2(元祖、本家、真打) ✅ 
- 妖怪ウォッチ3(スシ、テンプラ、スキヤキ) 🚧 (ストーリー5章までプレイ可能)

**対応リージョン**
- 日本 ✅
- 日本以外 ❌
